# 🛡️ Advanced Linux System Checkup Tool

A production-grade **Bash-based Linux auditing tool** that performs comprehensive system health, security, performance, and configuration analysis. The tool runs multiple audit modules and provides **color-coded PASS/WARN/FAIL results** along with an overall **System Health Score**.

> Perfect for System Administrators, DevOps Engineers, Security Analysts, Linux Enthusiasts, and Students.

---

## ✨ Features

- 🖥️ System Information Analysis
- ⚡ CPU Performance Monitoring
- 🧠 Memory & Swap Analysis
- 💾 Disk Usage & SMART Health Check
- 🌐 Network Diagnostics
- 🔒 Security Audit
- ⚙️ Service & Timer Analysis
- 📄 System Log Analysis
- 📦 Software Update Checker
- 🔥 Process Monitoring
- 🕒 Time Synchronization Check
- 🛡️ Kernel Hardening Verification
- 🐳 Docker Container Analysis
- 📊 Overall System Health Score
- 🎨 Color-coded PASS / WARN / FAIL Output

---

# 📋 Audit Modules

| Module | Description |
|---------|-------------|
| 🖥️ System Information | OS, Kernel, Architecture, Uptime, Boot Time, Logged Users |
| ⚡ CPU Analysis | CPU Model, Cores, Load Average, CPU Usage, Temperature, Governor |
| 🧠 Memory Analysis | RAM Usage, Swap Usage, HugePages, Top Memory Processes |
| 💾 Disk Analysis | Filesystem Usage, Inodes, SMART Health, I/O Statistics, Large Files |
| 🌐 Network Analysis | Interfaces, Gateway, DNS, Firewall, Open Ports, Connectivity |
| 🔒 Security Audit | SSH Config, UID 0 Users, Empty Passwords, SUID/SGID Files, Linux Capabilities |
| ⚙️ Services | Running Services, Failed Services, Timers, Cron Status |
| 📄 Log Analysis | Journal Errors, Log Sizes, Log Rotation |
| 📦 Updates | Pending Updates, Kernel Version, Reboot Status |
| 🔥 Process Analysis | Zombie Processes, High CPU/Memory Usage |
| 🕒 Time Sync | NTP, Chrony, RTC |
| 🛡️ Kernel Hardening | ASLR, NX, AppArmor/SELinux, ptrace, Core Dumps |
| 🐳 Docker | Running Containers & Images |
